Simulation code for *"Substrate Ontology V6.1: Chirality, Handedness, and Three-Fold Breaking"* (Zhou, 2026).

## Overview

This repository reproduces the key predictions of V6.1:
- **Parity restoration in weak interactions** at high energies (Section 5.1)
- **Temporary parity violation in strong interactions** under extreme density (Section 5.2)
